# 🖥️ System Monitor

Веб-дашборд для мониторинга системы в реальном времени. Показывает загрузку CPU, память, диск, сеть и топ процессов.

---

## ✨ Возможности

- ⚡ **Загрузка CPU** — общая и по каждому ядру
- 📈 **График истории CPU** — последние 30 секунд
- 🧠 **Оперативная память** — всего, доступно, процент использования
- 💾 **Диск** — всего, занято, свободно, процент заполнения
- 🌐 **Сеть** — отправлено и принято в MB
- 📋 **Топ-5 процессов** по загрузке CPU
- 🔄 **Real-time обновление** через WebSocket без перезагрузки страницы
- 🗑️ **Kill Process** — завершение процессов прямо из дашборда
- 📊 **История метрик** — сохранение в SQLite и график за последние 60 минут
- 💿 **Все диски** — отображение всех разделов (C:, D: и т.д.)
- 🔔 **Уведомления** — предупреждения при загрузке CPU или памяти выше 90%
- ⏱️ **Uptime** — время работы системы без перезагрузки
- 📡 **Сетевые соединения** — активные подключения (аналог netstat)
- 📋 **Автозагрузка** — список программ в автозагрузке Windows
- 🧵 **Потоки процессов** — количество потоков для каждого процесса
- 📶 **График сети** — история входящего и исходящего трафика
- 🔍 **Детали процесса** — подробная информация по клику (путь, память, пользователь)
- 📄 **Экспорт отчёта** — сохранение текущего снапшота в HTML-файл
- 📱 **Адаптивный дизайн** — поддержка планшетов и телефонов
- 🕐 **История за 24 часа** — переключение между часом и сутками
- 🎮 **GPU мониторинг** — температура, загрузка, видеопамять (NVIDIA)
- 💾 **SMART дисков** — модель, размер, статус здоровья
- 🖴 **Disk IO** — скорость чтения/записи для каждого диска
- 🔋 **Батарея** — заряд и статус (для ноутбуков)
- 📊 **Swap/Pagefile** — использование файла подкачки
- 🛡️ **Сканер портов** — список слушающихся портов на localhost
- 👤 **Активные пользователи** — кто залогинен в системе
- 📜 **Windows Event Log** — последние ошибки и предупреждения
- 🔔 **Настраиваемые триггеры** — уведомления при превышении порогов CPU/памяти
- 🎛️ **Панель управления виджетами** — включение/выключение отображаемых карточек
- 📈 **График Disk IO** — история чтения/записи в реальном времени
- 🗄️ **Автоочистка БД** — удаление записей старше 7 дней
- 📊 **Статистика** — средние/макс/мин значения за час или сутки
- ⚡ **Приоритет процесса** — изменение приоритета (Idle, Normal, High, Real-time)
- 🌳 **Дерево процессов** — родительские и дочерние процессы
- 🚀 **Запуск программы** — запуск программ прямо из дашборда

## 🛠️ Технологии

| Технология | Назначение |
|------------|------------|
| **Python 3.12** | Язык разработки |
| **FastAPI** | Веб-фреймворк и WebSocket |
| **psutil** | Сбор системных метрик |
| **Jinja2** | HTML-шаблоны |
| **Chart.js** | Графики на фронтенде |
| **Poetry** | Управление зависимостями |
| **SQLite** | Хранение истории метрик |
| **WMI** | SMART-данные дисков (Windows) |
| **nvidia-smi** | Мониторинг GPU NVIDIA |
| **pywin32** | Доступ к Windows Event Log |

## 🚀 Установка и запуск

### Предварительные требования

- Python 3.12 или выше
- [Poetry](https://python-poetry.org/docs/#installation)

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/your-username/system-monitor.git
cd system-monitor

# Установка зависимостей
poetry install
```

### Запуск

```bash
poetry run uvicorn app.main:app --reload
```
Открыть в браузере: http://127.0.0.1:8000

---

### API Эндпоинты

| Метод | Путь | Описание |
|------------|------------|------------|
| **GET** | `/` | Главная страница дашборда |
| **GET** | `/api/health` | Все метрики системы в JSON |
| **GET** | `/api/process/{pid}/info` | Детальная информация о процессе |
| **GET** | `/api/export` | Экспорт отчёта в HTML |
| **GET** | `/api/history?range=hour` | История метрик (hour/day) |
| **WebSocket** | `/ws/metrics` | Real-time поток метрик (обновление раз в секунду) |
| **GET** | `/api/triggers` | Получить настройки триггеров |
| **PUT** | `/api/triggers` | Обновить настройки триггеров |
| **GET** | `/api/widgets` | Получить настройки виджетов |
| **PUT** | `/api/widgets` | Обновить настройки виджетов |
| **GET** | `/api/stats?range=hour` | Статистика за период (hour/day) |
| **PUT** | `/api/process/{pid}/priority` | Изменить приоритет процесса |
| **GET** | `/api/process/{pid}/tree` | Дерево процессов (родитель + дочерние) |
| **POST** | `/api/run` | Запустить программу |
